# Cómo ejecutar la aplicación en local

## Preparación

- Descomprimir la actividad
- Asegurarse de tener instalado el instalador de paquetes `pnpm`
- Ejecutar dos terminales, una apuntando al directorio `frontend` y otra al directorio `backend`

En cada terminal, ejecutar:

```bash
pnpm install
```

## Arrancar los servidores

Una vez instaladas las dependencias de ambos directorios:

- **Backend** → `pnpm start`
- **Frontend** → `pnpm run dev`

Acceder a http://localhost:5173/ para utilizar la aplicación.
